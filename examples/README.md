## Using Examples

1. **Copy to your config directory**:
   ```bash
   cp examples/custom-profile.json ~/.config/nvim/profile.json
   ```

2. **Modify to your needs**:
   Edit the copied file to match your requirements

3. **Test the configuration**:
   ```bash
   nvim
   :NvimUltimateHealth
   ```
