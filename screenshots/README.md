## Contributing Screenshots

Want to contribute screenshots? Please:

1. Use high-quality screenshots (at least 1920x1080)
2. Use PNG format for best quality
3. Include captions/descriptions
4. Follow naming convention: `feature-name-description.png`
5. Submit via Pull Request

See [CONTRIBUTING.md](../CONTRIBUTING.md) for details.
