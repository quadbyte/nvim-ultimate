### 1. **Profile-Based Architecture**
Switch between 7 optimized profiles instantly:
- **Minimal**: Lightweight, blazing fast (<50ms startup)
- **Frontend**: JavaScript/TypeScript, React, Vue, Tailwind
- **Backend**: Python, Go, Rust, Node.js, SQL
- **Mobile**: React Native, Flutter, Swift, Kotlin

- **Startup Time**: <100ms even with full stack profile

- **11+ LSP Servers**: Full language server protocol support
- **10+ Themes**: 25+ variants for every preference
- **3 Icon Systems**: VSCode Codicons, Nerd Fonts, or Minimal ASCII
- **9 Modular Components**: Toggle features per profile

### 4. **Multi-Platform Support**
- macOS (ARM64 & Intel)
- Linux (Ubuntu, Arch, Fedora, Debian)
- Windows WSL2
